# OpenSpec Apply Workflow
Implement the approved change defined in `openspec/changes/<change-id>/`.
1. Read `proposal.md` and `tasks.md`.
2. Follow the implementation checklist in `tasks.md`.
3. Update the checklist as tasks are completed.
4. Ensure all new requirements and scenarios in delta specs are fully implemented.
