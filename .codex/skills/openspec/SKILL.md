# OpenSpec Skill
Understand and maintain specifications in `openspec/specs/` and changes in `openspec/changes/`.
Always refer to `openspec/AGENTS.md` for the 3-stage workflow.

## Workflows
- **Proposal**: Create new changes under `openspec/changes/`.
- **Apply**: Implement the changes defined in `proposal.md` and `tasks.md`.
- **Archive**: Archive the change into `openspec/changes/archive/`.
