# Project Context: El Decano Cobranzas

## Purpose
Integral management system for El Decano club: member management, membership fee collections, income/expenditure accounting, and advertising panel administration.

## Tech Stack
- **Frontend**: React 19 (Vite)
- **Backend/DB**: Supabase (@supabase/supabase-js)
- **Language**: TypeScript
- **Styles**: Tailwind CSS
- **Visualisation**: Recharts
- **Utilities**: XLSX (Excel import), Lucide React (Icons)

## Domain Context
1. **Members**: Management of members, categories, and bulk importation.
2. **Collections**: Fee recording, receipt issuance, and statements of account.
3. **Advertising**: Panel management (including 1.5-size panels) and advertisers.
4. **Accounting**: Financial movement recording by categories.

## Project Conventions
- **UK English** for code and technical documentation.
- **US English** for end-user facing content.
- **Spec-driven development** using OpenSpec framework.
- **Changes** proposed via specs in `openspec/changes/`.
